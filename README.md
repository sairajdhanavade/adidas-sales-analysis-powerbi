# 👟 Adidas US Sales Analysis (Power BI)

## 📌 Project Overview
This Power BI project analyzes Adidas sales data across the United States (2020-2021). The goal is to provide actionable insights into sales performance, regional trends, and product profitability to help stakeholders make data-driven decisions.

## 📊 Dataset Description
The dataset contains over 9,000 records of Adidas sales, featuring:
* **Retailers:** Foot Locker, Walmart, Sports Direct, etc.
* **Geography:** Region, State, and City.
* **Product Categories:** Men's/Women's Apparel and Footwear.
* **Financials:** Price per Unit, Units Sold, Total Sales, Operating Profit, and Operating Margin.
* **Sales Methods:** In-store, Online, and Outlet.

## 🛠️ Tech Stack & Tools
* **Power BI Desktop:** For data modeling and visualization.
* **Power Query:** For data cleaning and transformation.
* **DAX (Data Analysis Expressions):** For creating calculated measures.

## 📈 Key Insights & Dashboard Features
* **Sales Performance:** Total Sales and Operating Profit trends over time.
* **Product Analysis:** Identifying top-performing products and categories.
* **Regional Insights:** A map-based view of sales density across US States and Cities.
* **Sales Channel Efficiency:** Comparing Profit Margins between Online, Outlet, and In-store sales.

## 🚀 How to View
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. View the `Adidas_sales_report_recording.mp4.zip/` folder for a quick preview of the dashboard.
